Naming & Data Type Rules

Tag Name Conventions

  • TitleCase with no spaces, underscores, or special characters
  • Use Left and Right instea od East, West, North or South
  • Put the deliniation at the end FormerHeight no Former1Height
  • Be descriptive but succing: Blender SpeecHzCL2 not CLBlenderSpeed

Bit data type is not permitted

BITs cause conflicting display pattern sin downstream applications. Use INT instead
